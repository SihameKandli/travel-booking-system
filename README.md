# Travel Booking System

Applicazione web fullstack per la gestione di viaggi e prenotazioni.

## Tecnologie

- Node.js
- Express
- MySQL
- JavaScript

## Funzionalità

- Login e registrazione utenti
- Gestione viaggi
- Prenotazioni
- Autenticazione JWT

## Avvio progetto

1. `npm install`
2. Avvia database MySQL
3. `node server.js`
